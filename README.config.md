- Yarn使用		
版本锁定，缓存机制，安装速度快  	
- Yarn安装  
npm install yarn -g		

yarn init  === npm init 
yarn == npm install  
yarn global add === npm install -g   
yarn add   === npm install --save    
yarn add --dev === npm install --save-dev    
yarn remove  === npm uninstall --save(-dev)     
yarn run xxx === npm run xxx
